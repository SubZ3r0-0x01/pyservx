About PyServeX

PyServeX is a lightweight and easy-to-use HTTP server designed for quick file sharing within your local network. It features a unique retro-themed user interface, making file browsing and management a nostalgic experience.

Key features include:

  • File and folder browsing with a retro "hacker" UI.
  • Download entire folders as ZIP files.
  • Upload files via the web interface.
  • Accessible via localhost and network IPs.
  • QR Code for Easy Access from mobile devices.
  • Real-time Progress Bars for uploads and downloads.
  • Multiple File Uploads.
  • No File Size Restriction.
  • Enhanced Web Interface with Search Functionality and File Sorting.
  • Automated robots.txt for privacy.

Get started today and simplify your local file sharing!
